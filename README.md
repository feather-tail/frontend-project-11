## RSS Aggregator:

RSS aggregator to quickly get RSS feeds and read news.

## Install

#### install dependencies
``make install``

## Minimum requirements

Node: 21.6.1

npm: 10.9.2

## Commands

#### clear dist directory
``make remove``

#### build the project:
``make build``

#### start the Webpack Development Server:
``make develop``
